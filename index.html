What is Figuroner?

Figuroner is a small web app where you can enter your grades (and its grade weight). It's a school project and it's written entirely in HTML, CSS and JavaScript, built on top of Foundation (by ZURB). It uses GSAP for all animations.

Currently, the display language is Dutch.
